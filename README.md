To deploy the application using Kubernetes, follow these steps:

## Prerequisites

- Git (https://git-scm.com)
- Docker (https://www.docker.com)
- Kubernetes cluster (local, cloud provider, or managed Kubernetes service)

## Deployment Steps

1. Clone the Repository:
   
   git clone https://github.com/your-username/sit737-2023-t2-prac7.1p.git
   cd sit737-2023-t2-prac7.1p
</code></div></div></pre>

2. Set Up a Kubernetes Cluster:
   * Choose one of the following options to set up a Kubernetes cluster:
     * Local Cluster: Use a tool like Minikube or Docker Desktop to set up a local Kubernetes cluster.
     * Managed Kubernetes Service: Use a managed Kubernetes service like Google Kubernetes Engine (GKE), Amazon Elastic Kubernetes Service (EKS), or Azure Kubernetes Service (AKS).

   * Ensure that Docker is installed on your system.
   * Build the Docker image by running the following command:

   * Push the Docker image to a Docker registry accessible from your Kubernetes cluster.
